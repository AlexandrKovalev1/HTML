ТЕГИ ССЫЛОК
тег <a> тег ссылки
     атрибут href="" - адресс сылки (<a href="">текст ссылки </a> )
     атрибут target="" - казывает место (фрейм или окно браузера) в которое должен быть загружен, указанный в гиперссылке документ.
        target="_blank" открытие ссылки в новом окне (<a target="_blank" href="">текст ссылки </a> )
     атрибут download - атрибут для скачивания (не имеет значения)(<a href="">текст ссылки download текст ссылки </a> )

     АТРИБУТ id="" - уникальный идентификатор тега должен быть всегда уникальным # символ id
